## ‚è±Ô∏è 1. Countdown Timer

- User selects a date and time.
- If the selected date is in the future, the **Start** button becomes active.
- Clicking the Start button begins a countdown in days, hours, minutes, and
  seconds.
- When the countdown ends, the user receives a notification via **iziToast**.
- **flatpickr** library is used for cross-platform date/time selection.
- Initially, the Start button is disabled and becomes disabled again if an
  invalid date is selected.
